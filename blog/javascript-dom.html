Control flow and loops

Computers are… pretty dumb and obedient. Without giving any control flow, the computer will execute instructions, one after another, without waiting or hesitation. There will be no control as to what direction the code executes. Just one line. Then the next.

Control flows give order and direction to how a scope of code will execute:


Let's make a cup of tea.


  1. Fill Kettle with Water
  2. Turn the kettle on
  3. While waiting:
  4. If cup is not yet prepared, then prepare cup and place on your weird HTML/CSS table
  5. Else: Wait patiently for water to boil
  6. Once water has boiled and cup is prepared, pour water into cup
  7. Wait 2.5 minutes
  8. Remove Tea Bag
  9. Drink

In a similar frame, looping is a concept that relies on the computer's obedience. They will do whatever you ask for them, as long as you ask them, until you ask them to stop. There are different types of loops, such as Do While, ForEach, For loops and Recursion among others.

While (tea is not empty) {
Take a sip of Tea
Wait 2 seconds
};

next step

This would instruct the computer to take a sip of tea, wait 2 seconds, take a sip of tea, wait 2 seconds, and repeat until the cup is drained.

The exit condition is the cup becoming empty. Only then will the program move on to the next step.

Describe What the DOM is and an example of how you might interact with it

The DOM stands for 'Document Object Model'. In this instance we will be referring to the HTML DOM. This is a logical structure that lays out how the HTML elements will be loaded onto a web page. The layout and information contained in the program code will give instruction as to how this particular DOM will be formed, such as what elements it will contain, updating and changing of the elements within the DOM. Explain the differences between accessing data from Arrays and Objects

Objects and Arrays are both ways to store collections of data values. In terms of behavior, arrays behave very similar to lists. Lets say you wanted to have a collection of numbers: 1,2, 3 ,4 and 5.

You could group these together in an Array Variable. Arrays have a set of square brackets surrounding the structure of data that is held within the array, to group them together.

This is stored as a single array object, which you can access the indexes of, to reach a single value stored within the array.

let numbersArray = [1,2,3,4,5];

Objects on the other hand, store data with key:value pairs. This provides flexibility, and for multiple pieces of information to be stored within the same variable.

Let person = { name : ‘Lawrence’, Age: 27, LikesCoffee: true }

This allows for far richer attributes of information attached, such as important details making up a person object.

Explain what functions are and why they are useful

Functions are like little sets of instructions or mini-programs within your code block to do something.

This is a great way to add power and logic into your program, such as a function to add two numbers together that were generated somewhere else in your code. This allows better re-usability and the ability to better organize and make your code more modular.
Initially, a function is declared.


Function addTwoNumbers(a,b) { Return a + b; }

Then, the function is called.


addTwoNumbers(2,4) // outputs 6

The advantage of this modular instruction being tucked away into its own function is that addTwoNumbers could be called multiple times in different parts of a program, as required.
